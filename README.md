### Secrets
Create a file `.env` at the project root and add the following to it:
```
REACT_APP_NYT_API_KEY=<NY Times API key>
REACT_APP_RAPIDAPI_KEY=<RapidAPI key>
```

### Package
```
yarn package
```
Run `Today's Headlines Setup 0.0.1` under the `release` directory.

### Run in dev
```
yarn start
```
